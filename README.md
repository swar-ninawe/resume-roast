# 🔥 ResumeRoast

**Upload your resume. Let AI roast it.**

ResumeRoast is a brutally honest AI-powered resume reviewer that analyzes your resume like a no-nonsense recruiter. It highlights weak points, destroys buzzwords, and suggests improvements to help you build a stronger, job-ready resume.

🌐 **Live Demo:**
[https://resume-roast-roan.vercel.app/](https://resume-roast-roan.vercel.app/)

---

# 🚀 Features

### 🔎 ATS Score Analysis

Simulates how Applicant Tracking Systems evaluate your resume and gives a rough **score based on structure and keywords**.

### 🧠 Brutal AI Feedback

An AI recruiter reviews your resume and provides **honest feedback** about:

* Weak bullet points
* Overused buzzwords
* Poor formatting
* Lack of measurable impact

### 💣 Buzzword Detection

Identifies generic resume phrases like:

* “Hardworking”
* “Team player”
* “Results-oriented”

…and suggests stronger alternatives.

### ✍️ Bullet Point Improvements

AI rewrites weak bullet points into **impact-driven, recruiter-friendly statements**.

### 🎨 Modern UI

Clean dark interface with a minimalist design for a smooth experience.
